Q&A Discussion

Q. How do you ensure only pictures of rooms are taken?
Using the backend integration with Machine Learning, we are able to detect all sorts of items that range from typical room (e.i. lamp, thermostat, bed, etc.). If there are any items (e.i. a face), our code will put out an error message to prompt the user to input appropriate images.

Q. Where does Leaf's data come from?
We spent a large portion of our time ensuring that we provide 100% factual information to our users as seen in the research phase. Because all of the information we provide is from publically available sources, there should be no significant issues with data breaches. As for the images our users submit, we ensure that it will be kept private and only for the use of the users via encryptions.
